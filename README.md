**How to Install**

==============

**for Android**

**pkg install git -y**

**pkg install python -y**

**pip install requests**

**git clone https://github.com/kolllmar2/cpm.git**

**cd cpm**

**python cpm1.py for cpm1**

**python cpm2.py for cpm2**


=============

**for IOS**

**apk add git**

**apk add python3**

**apk add py3-pip**

**pip install requests**

**git clone https://github.com/kolllmar2/cpm.git**

**cd cpm**

**python3 cpm1.py for cpm1**

**python3 cpm2.py for cpm2**
